# SRT Audio Streamer - クイックスタートガイド

すぐにアプリを試せるように、最小限の手順をまとめました。

## 🚀 5分で始める

### 1. テストサーバーを起動

ターミナルを開いて実行:

```bash
cd /Users/haya2/github/srtto-sound-only-live
./start-test-server.sh
```

スクリプトが自動的に:
- FFmpegの有無をチェック
- MacのIPアドレスを表示
- SRT受信サーバーを起動

**出力例:**
```
MacのIPアドレス: 192.168.1.10

📱 iPhoneのアプリで以下のURLを入力してください:
   srt://192.168.1.10:9710?mode=caller
```

### 2. Xcodeでアプリをビルド

```bash
open SRTAudioStreamer.xcodeproj
```

1. Xcodeでプロジェクトが開く
2. 対象デバイス（iPhone）を選択
3. Product > Run (⌘R) を実行
4. 初回は依存関係のダウンロードに数分かかります

### 3. アプリを使う

#### 初回起動
1. アプリが起動したら、マイク権限を「許可」

#### 配信開始
1. **SRT URL** に、スクリプトが表示したURLを入力
   - 例: `srt://192.168.1.10:9710?mode=caller`
2. **ビットレート** は 64 kbps のまま（デフォルト）
3. **配信開始** ボタンをタップ

#### 確認
- 状態が「配信中」（緑色）になる
- ビットレート表示が更新される（約64.0 kbps）
- FFmpegのターミナルで音声データの受信が確認できる

```
Stream #0:0: Audio: aac, 44100 Hz, stereo, fltp, 64 kb/s
```

#### 配信停止
- **配信停止** ボタンをタップ

## ⚠️ トラブルシューティング

### FFmpegがない場合

```bash
brew install ffmpeg
```

### 接続できない場合

**チェック項目:**
1. MacとiPhoneが**同じWi-Fi**に接続されているか
2. IPアドレスが正しいか（Wi-Fi再接続後は変わることがある）
3. FFmpegが起動しているか
4. ファイアウォールでポート9710がブロックされていないか

**対処:**
```bash
# IPアドレスを再確認
ifconfig | grep "inet " | grep -v 127.0.0.1

# FFmpegを再起動
# Ctrl+C で停止 → ./start-test-server.sh で再起動
```

### マイクが使えない場合

**設定アプリを開く:**
1. 設定 > プライバシーとセキュリティ > マイク
2. "SRTAudioStreamer" をオンにする
3. アプリを再起動

### ビルドエラーが出る場合

**依存関係を再解決:**
1. Xcodeで File > Packages > Resolve Package Versions
2. 数分待つ
3. 再度ビルド

## 🎯 次のステップ

基本的な配信ができたら、以下を試してみましょう:

### ビットレートを変更

1. 配信停止
2. ビットレートを 96 kbps または 128 kbps に変更
3. 配信開始
4. ビットレート表示が変わることを確認

### 長時間配信

1. 配信を開始
2. 5分〜10分間継続
3. 安定して動作することを確認

### バックグラウンド配信

1. 配信を開始
2. ホームボタンを押してアプリをバックグラウンドに
3. 1分待つ
4. アプリに戻る
5. 配信が継続されていることを確認

## 💡 便利なコマンド

### FFmpegで音声をファイルに保存

```bash
ffmpeg -i srt://0.0.0.0:9710?mode=listener output.aac
```

### FFmpegで音声をリアルタイム再生

```bash
ffmpeg -i srt://0.0.0.0:9710?mode=listener -f audiotoolbox default
```

### ポートが使用中か確認

```bash
lsof -i :9710
```

## ❓ よくある質問

### Q: Simulatorで動作しますか？
A: シミュレーターにはマイクがないため、実機での使用を推奨します。

### Q: Wi-Fi以外で接続できますか？
A: 可能です。MacとiPhoneが同じネットワーク上にあれば接続できます。

### Q: インターネット経由で配信できますか？
A: このバージョンはローカルネットワーク用です。外部配信には公開SRTサーバーが必要です。

### Q: 音声品質を変更できますか？
A: ビットレートを変更することで調整可能です（64/96/128 kbps）。

### Q: バッテリー消費はどのくらいですか？
A: 配信設定によりますが、64 kbpsで約10-15%/時間程度です。
